# MBTI-WeChat-Mini-Program
只需一分钟就能测出你的性格特点
## React+Taro ##
使用Webstorm用Typescript语言进行程序的开发，通过调用taro ui组件库提高开发效率。
通过对简易的题目库和答案库的json文件进行检索，并进行各个人格类型的评分，选出最高分即为结果。
